# DevCamper API

> Backend API for DevCamper application which isa bootcamp directory website

## Usage

Rename "config/config.env.env" to "config/config.env" and update values/settings to your own

## Install Dependencies

```
npm install
```

## Run App

```
#Run in dev mode
npm run dev

#Run in prod mode
npm start
```

- Version: 1.0.0
